GAM 374 FUNDAMENTALS OF GAME PROGRAMMING I

Fundamental concepts of game programming using C/C++ and OpenGL. Students will learn how to implement simple 3D action games: 3D rendering operations including texture mapping, physical interactions including collisions, input / output management including game UI, analog controllers, and sound. Emphasis will be placed on the development of the coding skills needed for robust, efficient, and portable implementation. PREREQUISITE(S): (GAM 372 or SE 350) and GAM 325
